Question-1 : How does Javascript work ?

Javascript is a interpreted language which means that it's does not need to compile before run or execution. It works in just in time compiled. It handle one line code execution at a time. Javascript is a single threaded high level programming language. Javascript use v8 engine for run the code into web browser. The code execution process is synchronous way. and use single call stack. Allthings in a javascript happens inside an execution context.

Question-2 : Difference between localStorage and sessionStorage

The local storage data has no expiration date it's stored in the local storage , if we want to reuse it we can but the session storage is one stay memory for the current session, if the current session or current browser tab is closed than the session is deleted. Local storage is permanent and the session storage is temporary data. Data stored in session storage is specific to the protocol of the single page ,localStorage data is specific to the protocol of the whole document.
